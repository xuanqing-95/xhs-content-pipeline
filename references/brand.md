# 心象品牌风格

## 视觉规范

| 维度 | 规范 |
|------|------|
| 色调 | 深青绿、青灰绿、暖米色、柔和金黄 |
| 质感 | 纸张纹理、手绘感、水彩晕染 |
| 风格 | 治愈系、温暖、自然意象、正念 |
| 元素 | 山川、河流、月亮、花草、自然物 |
| 字体 | 手写感中文、细体英文辅助 |
| 背景 | 浅青绿色或米白（不要暖米色） |

## baoyu-xhs-images 风格映射

配图由 `baoyu-xhs-images` skill 生成，根据选题内容自动匹配风格，不写死。

### 默认匹配逻辑

| 选题类型 | 推荐风格 | 推荐 preset |
|---------|---------|------------|
| 健康/养生/自然/正念 | `fresh` | `nature-flow` |
| 情感/生活故事/温暖 | `warm` | `cozy-story` |
| 干货/方法论/知识 | `notion` | `knowledge-card` |
| 清单/避坑/提醒 | `notion` | `checklist` |
| 自动（默认） | 由 baoyu-xhs-images 根据内容信号自动判断 | — |

### 可手动切换的风格

用户可在确认配图方案时指定：`--style fresh`、`--style notion`、`--style warm` 等，
或直接说「用 XX 风格」，由 Step 4 传参覆盖。

### 品牌色彩建议（供生图参考）

- 主色调：深青绿 #2D6A6A、青灰绿 #4A7C7C
- 辅助色：暖米色 #F5F0E8、柔和金黄 #D4A853
- 背景：浅青绿 #E8F4F4 或米白 #FAFAF5
- 元素偏好：山川、月亮、花草、自然物、水彩晕染、手绘纹理

# 心象品牌风格

## 视觉规范

| 维度 | 规范 |
|------|------|
| 色调 | 深青绿、青灰绿、暖米色、柔和金黄 |
| 质感 | 纸张纹理、手绘感、水彩晕染 |
| 风格 | 治愈系、温暖、自然意象、正念 |
| 元素 | 山川、河流、月亮、花草、自然物 |
| 字体 | 手写感中文、细体英文辅助 |
| 背景 | 浅青绿色或米白（不要暖米色） |

## 配图 Prompt 模板

生成配图时，基于以下模板构建 prompt，根据具体选题替换 `{主题}` 和 `{核心元素}`。

### 封面图（P1）
```
治愈系插画风格，浅青绿色背景，水彩晕染质感。
画面中心是{核心元素}，周围点缀花草和自然意象。
整体氛围温暖宁静，带有手绘纸张纹理。
中文手写体标题「{主题}」居中偏上。
竖版 3:4 比例，小红书封面风格。
```

### 内容图（P2-P3）
```
治愈系信息图风格，米白色背景，纸张纹理质感。
用简约手绘图标展示{要点内容}。
配色：深青绿文字 + 柔和金黄点缀。
布局清晰，留白充足，易于阅读。
竖版 3:4 比例。
```

### 结尾图（P4）
```
治愈系插画，浅青绿渐变背景，水彩月亮或山川意象。
画面简洁温暖，传递正念和希望。
底部留空适合放置账号信息。
竖版 3:4 比例。
```

# 表格字段定义

## 选题库（tblBBoGrB5W6DfB4）

### 读取筛选

```json
{
  "conjunction": "and",
  "conditions": [
    {"field_name": "选题状态", "operator": "is", "value": ["待选"]}
  ]
}
```

### 状态流转

```
待选 → 已推送（Step 1 推送后）→ 已使用（Step 2 用户选中后）
```

### 数据解析注意

飞书返回的 `选题标题` 字段是数组格式：
```
fields["选题标题"][0].text  → 实际标题文本
```

## 小红书内容生成库（tblcdl9yD51wZHVt）

Step 5 写入，发布后手动或触发「回填数据」更新数据字段。

### 完整字段清单

| 字段 | 类型 | 说明 | 写入时机 |
|------|------|------|----------|
| 标题 | Text | 小红书文案标题 | Step 5 |
| 正文 | Text | 完整正文内容 | Step 5 |
| 标签 | Text | #标签1 #标签2 | Step 5 |
| 选题方向 | Text | 使用的创作方向 | Step 5 |
| 分类标签 | Text | 内容分类 | Step 5 |
| 配图链接 | Text | 飞书图片链接（逗号分隔） | Step 5 |
| 原文链接 | Text | 参考原文 URL | Step 5 |
| 创建时间 | DateTime | 草稿生成时间（毫秒时间戳） | Step 5 |
| 发布时间 | DateTime | 实际发布时间（毫秒时间戳） | 手动回填 |
| 发布状态 | SingleSelect | 待发布/已发布/已归档 | Step 5 写「待发布」，发布后改「已发布」 |
| 阅读量 | Number | 笔记阅读数 | 数据回填 |
| 点赞数 | Number | 点赞数 | 数据回填 |
| 收藏数 | Number | 收藏数 | 数据回填 |
| 评论数 | Number | 评论数 | 数据回填 |
| 分享数 | Number | 转发/分享数 | 数据回填 |
| 爆款评级 | SingleSelect | 自动计算：S/A/B/C/D | 数据回填时自动评级 |
| 复盘备注 | Text | 复盘时填写的经验总结 | 复盘时写入 |

### 爆款评级规则（数据回填时自动计算）

以点赞+收藏为核心指标：
```
点赞+收藏 ≥ 1000  → S（爆款）
点赞+收藏 500-999 → A（优秀）
点赞+收藏 200-499 → B（良好）
点赞+收藏 50-199  → C（普通）
点赞+收藏 < 50    → D（待优化）
```

### 注意事项

- URL 字段格式：`{"link": "url", "text": "描述", "type": "url"}`
- 正文中英文双引号 `"` 替换为中文 `""`（否则 JSON 解析失败）
- Step 5 写入时，「发布状态」默认填「待发布」

## 飞书上传 Workaround

`feishu_drive_file` upload 的 `folder_token` 参数不生效。

**正确流程：**
1. upload 到根目录（不传 folder_token）
2. 从返回结果获取 file_token
3. 用 `feishu_drive_file` move 到目标文件夹
4. upload 后立即 move，不要延迟
